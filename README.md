# 项目结构

```
flask_ocr/
├── checkpoints
├── detect
├── recognize
├── app.py
├── ocr.py
└── test.html
```



- checkpoints ：存放训练的网络模型
- detect ： 训练路径以及相关配置
- recognize：训练识别
- app.py：flask框架与前端交互的api
- ocr.py：识别接口
- test.html ： 测试网页
